# My Unity Game

A simple 3D endless runner game made with Unity.

## Features
- Player movement
- Obstacles
- Score system
- Game over and restart

## Controls
- A / D → Move left and right
- Space → Jump

## Made With
- Unity
- C#

## How to Play
1. Open the project in Unity
2. Open the Main Scene
3. Press Play
